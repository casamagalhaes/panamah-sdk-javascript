## Instalação

1. Execute o comando
    ```bash
    npm install panamah-sdk-javascript --save
    ```
2. Utilize as APIs e modelos através do 
    ```javascript 
    { PanamahStream, PanamahAdmin, PanamahModels } = require('panamah-sdk-javascript')
    ```
